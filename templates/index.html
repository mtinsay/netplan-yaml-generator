About Netplan Web Generator

Version: 1.0.0

Copyright: ¬© 2025 Michael Tinsay

License: GNU General Public License v3.0

Description

A standalone Go web application for generating netplan YAML configurations through a user-friendly web interface. Supports ethernet, bond, and bridge interfaces with both DHCP and static IP configuration options.

Features

  • Ethernet interface configuration (DHCP/Static)
  • Bond interface configuration (7 bonding modes)
  • Bridge interface configuration
  • DHCP override options
  • IPv4 and IPv6 support
  • Real-time YAML generation
  • Copy to clipboard functionality
  • Responsive web design
  • Automatic ethernet declarations for bond/bridge interfaces

Important Notes

Bond and Bridge Interfaces: When creating bonds or bridges, the underlying physical interfaces are automatically declared in the ethernets section with dhcp4: false to prevent IP address conflicts.
